💥 Mini résumé clair des commandes

git add .
git commit -m "V6"
git push origin main


✅ 1. Tu modifies tes fichiers dans VSC
→ par exemple style.css, index.html, etc.

✅ 2. Tu les ajoutes au suivi Git

git add nom_du_fichier
Exemple :

git add style.css

Ou tout d’un coup :

git add .

✅ 3. Tu fais un commit (tu “valide” le changement localement)
Modifier
git commit -m "ton message"
Exemple :
git commit -m "modifie le css pour le header"

✅ 4. Tu pousses sur GitHub (tu envoies vers le serveur distant)
git push origin main
